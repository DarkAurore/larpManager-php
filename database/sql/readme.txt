Ce repertoire ne doit contenir que des fichiers SQL nommes database_install.sql
Ces SQL sont generes depuis les fichiers mwb et permettent l'installation ou la mise a jour 
de la base afin de palier au manque de ligne de commande.
Ils ne sont utilises que pour la creation ou la mise a jour, et par l'administration.
Pour les executer, voir le controlleur install
Il faut nommer le fichier create_or_update.sql
